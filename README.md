
# 📊 Amazon Sales Power BI Dashboard  
## Business Analysis Project

## 📌 Overview
This project presents a comprehensive Business Intelligence analysis using a Power BI dashboard built on Amazon sales data. The dashboard visualizes key business metrics such as sales, profit, orders, customer behavior, product performance, and regional distribution to support data-driven decision making.

## 🎯 Project Objectives
- Analyze overall sales and profit performance  
- Identify top-performing products and categories  
- Evaluate regional sales distribution  
- Understand customer purchasing behavior  
- Generate actionable business insights

## 📊 Key Features

### 🔹 Key Performance Indicators (KPIs)
- Total Sales  
- Total Profit  
- Total Orders  
- Profit Margin  
- Customer Count  

### 🔹 Analytical Views
- Sales Trend Analysis (Time Series)
- Category-wise Performance
- Region-wise Sales Distribution
- Customer Insights
- Profitability Analysis

## 🧠 Business Insights
- A small number of categories drive most revenue.
- Urban regions generate the highest sales.
- Seasonal spikes occur during promotional periods.
- Heavy discounting increases sales but reduces profit margins.
- Repeat customers contribute significantly to revenue.
